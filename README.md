# homebrew-locksmith

Homebrew Tap for [Locksmith](https://github.com/bonjoski/locksmith).

## Installation

```bash
brew tap bonjoski/locksmith
brew install locksmith
```

## Verification

After installation, you can verify the version:

```bash
locksmith --version
```

## Documentation

For more information, see the [main repository](https://github.com/bonjoski/locksmith).
